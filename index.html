Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeather API to assemble a dataset of over 500 cities.

After assembling the data set, we used Matplotlib to plot various aspects of the weather vs latitude. factors we looked at included: temprature, cloufiness, humidity and wind speed. this site provides the source data and visualization created as part of the analysis as well as explanations and descriptipons of any trends and correlations witnessed.
